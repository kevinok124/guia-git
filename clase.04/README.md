## GIT TAG

## Listar tags

```sh
git tag
```

## Ver que se hizo y quien tagueo este tag

```sh
git show <nombre-tag>
```

## Crear un tag

```sh
git tag -a v1.0.0
git tag -a v1.0.0 -m "Versión 1.0.0" # -a:annotate | -m:message
```
